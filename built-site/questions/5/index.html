Notes

Aggregation

The regional gross disposable household income comes from the ONS/NOMIS at ITL3 geography level. We used the Open Innovations geography lookup builder along with ITL3 boundaries (ONS, 2021) and DNO license areas (National Grid) to calculate the fraction (by area) of overlap between each ITL3 region and each DNO license area. This was saved as JSON output and then used to apportion the ITL3-level data to DNO license areas.

Per head

The totals for gross disposable income and population in each DNO license area were added up separately and then the per head values calculated by dividing total income by total population.
